SB Insights RSS Feed Plugin

== Description ==

This plugin enables a widget that pulls a RSS Feed from the Samsung Business Insights Blog (http://insights.samsung.com/).

The plugin allows you to choose from 3 different feed category sources:

* Display
* Mobility
* Storage 

You can control for how long the feed should stay in cache as well.

== Installation ==

1. Download the zip file
2. Uncompres and upload the folder `insights-samsung-feed` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the widget to any sidebar you have in your theme
